## Featured

### Hércules — Fiscal Execution System
*Private · Python · FastAPI · PostgreSQL*

A comprehensive fiscal execution management system built for government accountability workflows. Handles budget tracking, commitment generation, payment scheduling, and reporting across multiple fiscal years. Integrates with SIAFI (Brazil's federal accounting system) APIs.

**Tags:** `fiscal-tech` `government` `python` `fastapi` `postgresql`

---

### OLIM — NLP Annotation & Labeling Tool
*Private · Python · React · FastAPI*

An internal web application for annotating and labeling datasets for NLP model training. Features include multi-user annotation, agreement metrics (Cohen's kappa), label versioning, and export to Hugging Face datasets format.

**Tags:** `nlp` `annotation` `react` `fastapi` `machine-learning`

## Open Source

### [CantinaCF](https://github.com/felipeadeildo/CantinaCF)
A digital canteen management system built for a school environment. Handles orders, payments, and inventory. Built with Django + HTMX for a fast, reactive UI without heavy JavaScript.

**Tags:** `django` `htmx` `python` `web`

---

### [AutoSinan](https://github.com/felipeadeildo/AutoSinan)
Automation tool for SINAN (Brazil's notifiable diseases information system). Automates data entry, form filling, and report generation for public health surveillance workflows.

**Tags:** `automation` `selenium` `python` `public-health`

---

### [Gestão Oncológica](https://github.com/felipeadeildo/gestao-oncologica)
A web system for managing oncology patient records and treatment schedules. Built during research at NES/Insper.

**Tags:** `healthcare` `django` `python` `research`

### [whatsapp-mcp](https://github.com/felipeadeildo/whatsapp-mcp)
A Model Context Protocol (MCP) server that exposes WhatsApp functionality to LLM agents. Allows AI assistants to send messages, read chats, and manage contacts via WhatsApp Web.

**Tags:** `mcp` `llm` `whatsapp` `typescript` `ai`

---

### [neural-network](https://github.com/felipeadeildo/neural-network)
A neural network implementation from scratch in Python — no PyTorch, no TensorFlow. Educational project covering forward/backward pass, gradient descent, and layer abstractions.

**Tags:** `machine-learning` `numpy` `python` `educational`

---

### [pytally-sdk](https://github.com/felipeadeildo/pytally-sdk)
Unofficial Python SDK for the [Tally](https://tally.so) forms API. Simplifies form creation, submission fetching, and webhook handling.

**Tags:** `sdk` `python` `api` `tally`
